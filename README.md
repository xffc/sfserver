<h1 align="center">Simple File Server</h1>

## 📋 Setup and usage
1. Download sfs <a href="https://github.com/xffc/sfserver/releases">here</a>.
2. Unpack it
3. Launch SFileServer.exe
4. Enter port (I recommend to use 80 or 5050)
5. Get a link to server
